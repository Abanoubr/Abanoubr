### What I'm building

**[ProTeach](https://github.com/Abanoubr/proteach-app)** — A full-stack homeschool platform with personalized lesson plans, educational games, and real-time progress tracking for families across all 50 US states.

<table>
<tr>
<td width="50%">

**Platform Features**
- Personalized weekly lesson plans (state DOE-aligned)
- Progressive educational game engine with XP & badges
- Parent, Student, and Admin portals
- Real-time progress tracking & attendance
- AI-powered game question generation
- Stripe subscription management

</td>
<td width="50%">

**Tech Stack**
- Next.js 16 (Turbopack)
- TypeScript &bull; Tailwind CSS v4
- Firebase (Auth + Firestore + Storage)
- Framer Motion (animations)
- Google Gemini AI (internal)
- Stripe &bull; shadcn/ui &bull; Web Audio API

</td>
</tr>
</table>
